# CannaRoute — Customer App

React Native (iOS + Android)

## Screens
- Onboarding / Age verification / Medical card flow
- Home — Dispensary discovery map + list
- Dispensary menu — Product catalog with grower transparency badges
- Product detail — Full grower profile, pesticide log, COA results, batch traceability
- Cart + Checkout — Compliance limit check, payment method selection, tax breakdown
- Live order tracking — Real-time GPS map, driver info, status steps
- Order history
- Profile / Settings

## Key Dependencies (planned)
- `react-native` — core framework
- `@react-navigation/native` — navigation
- `react-native-maps` — Google Maps SDK
- `socket.io-client` — real-time order tracking
- `@stripe/stripe-react-native` — payment UI
- `react-native-camera` — ID scan at onboarding + driver ID check
