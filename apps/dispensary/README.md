# CannaRoute — Dispensary Dashboard

React.js (web)

## Screens
- Overview — Real-time metrics, active order queue, driver status, revenue chart
- Order detail — Compliance check results, driver assignment, manifest generation
- Inventory / Menu — Product catalog, grower linking, COA status, Metrc sync
- Driver management — On-duty drivers, GPS positions, performance
- Compliance — Delivery manifests, Metrc sync logs, audit trail
- Analytics — Sales by product/category/time, delivery performance, customer data

## Key Dependencies (planned)
- `react` + `vite` — core framework + build
- `react-query` — server state / data fetching
- `socket.io-client` — real-time order updates
- `@googlemaps/react-wrapper` — driver map view
- `recharts` — analytics charts
- `react-table` — order and inventory tables
